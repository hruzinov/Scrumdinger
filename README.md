# Scrumdinger

A training pet-project where a step-by-step iOS app is created for the Scrum technique.

![](screenshot.jpg)

## Tech Stack

**Code:** Swift

**UI:** SwiftUI

**Additional:** Microphone access and speech recognition

*Min. version: iOS 15.0*
